# Task 2
Determine the first derivative of the softplus function which is defined as $f(x) = \ln(1 + e^x)$.

# Solution
TO find the first derivitive we can use the chain rule.

The chain rule states that if you have a composite function $f(g(x))$, its derivative is $f'(g(x)) \cdot g'(x)$.

The outer function is $\ln(u)$, where $u = 1+e^x$. The derivitive of $\ln(u)$ is $\frac{1}{u}$.
The inner function is $u = 1 + e^x$ has the derivitive is $e^x$.

COmbining the derivitives we therefore have, 

$$f'(x) = \frac{1}{1 + e^x} \cdot \frac{d}{dx}(1 + e^x)$$

$$f'(x) = \frac{1}{1 + e^x} \cdot e^x$$

Final Simplified Form$$f'(x) = \frac{e^x}{1 + e^x}$$

Note: If you divide both the numerator and the denominator by $e^x$, you get a very famous result in machine learning:$$f'(x) = \frac{1}{\frac{1}{e^x} + 1} = \frac{1}{1 + e^{-x}}$$This is the Sigmoid (logistic) function.
