# Task 1
The hyperbolic tangent is one of the choices for an activation function. Determine the first derivative of the hyperbolic tangent. Notation:  

$$\tanh(x) = \frac{e^x-e^{-x}}{e^x+e^{-x}}$$

# Solution
To find the first dervitive we can use the **quotient rule**, that is

$$\frac{d}{dx} \left( \frac{u}{v} \right) = \frac{u'v - uv'}{v^2}$$

Therefore,

$$\frac{d}{dy}\tanh(x) = \frac{uv'- vu'}{v^2} $$
and,

$$u = e^x-e^{-x}$$
$$v = e^x+e^{-x}$$

With the rule that the derivitie of $e^x$ is $e^x$ nd the dervitive od $e^{-x}$ is $-e^{-x}$.

$$u' = e^x+e^{-x}$$
$$v' = e^x-e^{-x}$$

Which reduces to 

$$\frac{d}{dx} \tanh(x) = \frac{4}{(e^x + e^{-x})^2}$$
