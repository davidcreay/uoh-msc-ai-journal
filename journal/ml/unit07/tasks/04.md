# Task 4

Calculate the first derivative $\frac{\delta{H}}{\delta{q_{i}}}$ of the cross-entropy loss function $H(p,q)=\sum^{d}_{i=1} p_{i}log(q_{i})$.

# Solution
In machine learning, **Cross-Entropy Loss** (also called Log Loss) is a performance metric used to measure how well a classification model's predicted probability distribution matches the actual "ground truth" labels.

It is the standard loss function for **multi-class classification** problems.

---

### 1. The Concept

Imagine you are training a model to recognize animals. If the true label is "Cat," but your model is only 20% sure it's a cat (giving it a probability of 0.2), Cross-Entropy Loss will assign a high penalty. If the model is 95% sure (0.95), the loss will be very low.

The goal of the training process is to **minimize** this loss.

---

### 2. The Formula

For a single observation, the formula is:


$$H(p, q) = -\sum_{i=1}^{d} p_i \ln(q_i)$$

Where:

* $p_i$: The **true** probability (usually $1$ for the correct class and $0$ for others).
* $q_i$: The **predicted** probability from the model (e.g., from a Softmax layer).
* $d$: The total number of classes.

---

### 3. How it Works (The "Log" Logic)

The use of the natural logarithm ($\ln$) is what makes the function effective:

* **When the prediction is correct ($q_i \approx 1$):** $\ln(1) = 0$, so the loss is near zero.
* **When the prediction is wrong ($q_i \approx 0$):** As $q_i$ approaches zero, $\ln(q_i)$ becomes a very large negative number. The negative sign in front of the formula turns this into a **massive positive penalty**.

---

### 4. Binary vs. Multi-class

* **Binary Cross-Entropy (BCE):** Used for "Yes/No" tasks (e.g., Spam vs. Not Spam).

$$L = -[y \ln(\hat{y}) + (1-y) \ln(1-\hat{y})]$$


* **Categorical Cross-Entropy:** Used when you have 3 or more categories (e.g., Dog, Cat, Bird).

---

### Why use this instead of simple Accuracy?

Accuracy is "all or nothing"—it doesn't tell the model *how far off* it was. Cross-Entropy provides a **smooth gradient**, telling the model: "You were wrong, and here is exactly how much you need to adjust your weights to be more confident next time."
