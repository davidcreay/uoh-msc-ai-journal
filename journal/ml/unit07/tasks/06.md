# Task 5
A classifier has been built that computes $\hat{p}$ as a predicted probability for a material to pass a stress test, given the real-valued properties $x_1$ and $x_2$ of the material. The classifier’s internal calculations are: 

$z = \alpha x_1 + x_2$ 

$h_1 = \arccos(z)$

$h_2 = \ln(1 + e^{\beta x_2 + \gamma})$ 

$u = h_1 - 3h_2 + \delta$ 

$\hat{p} = \frac{e^u}{e^u + 1}$ 

$\alpha, \beta, \gamma, \delta$ are internal tunable parameters of the classifier. Write out the partial derivatives of the sum of squared error loss function and the cross-entropy loss function with respect to the tunable parameters, given a training data point $(x_1, x_2, p)$.

# Solution 

The trick is to look at each line as a **step in a recipe**. Each line tells you how one variable is built using the ones before it. To find the derivative, we just ask: *"If I nudge this specific parameter, how much does the result of this line change?"*

Here is the step-by-step breakdown of how those components were obtained:

### 1. The Input Layer ($z$)

The first equation is $z = \alpha x_1 + x_2$.

* **To find the $\alpha$ component:** We differentiate $z$ with respect to $\alpha$.
* Since $x_1$ is just a multiplier (like a constant), the derivative is simply: **$\frac{\partial z}{\partial \alpha} = x_1$**.

### 2. The First Hidden Path ($h_1$)

The second equation is $h_1 = \arccos(z)$.

* This uses the standard calculus rule for the derivative of the inverse cosine.
* The derivative of $\arccos(z)$ is: **$\frac{\partial h_1}{\partial z} = -\frac{1}{\sqrt{1-z^2}}$**.

### 3. The Second Hidden Path ($h_2$)

The third equation is $h_2 = \ln(1 + e^{\beta x_2 + \gamma})$.

* This is the **Softplus** function we calculated in Question 2!
* **For $\beta$:** We use the chain rule. The derivative of $\ln(1+e^u)$ is the Sigmoid function $\frac{e^u}{1+e^u}$. Then we multiply by the derivative of the inside ($\beta x_2 + \gamma$) with respect to $\beta$, which is $x_2$.
* **$\frac{\partial h_2}{\partial \beta} = \frac{e^{\beta x_2 + \gamma}}{1 + e^{\beta x_2 + \gamma}} \cdot x_2$**.


* **For $\gamma$:** It's the same logic, but the derivative of $(\beta x_2 + \gamma)$ with respect to $\gamma$ is just $1$.
* **$\frac{\partial h_2}{\partial \gamma} = \frac{e^{\beta x_2 + \gamma}}{1 + e^{\beta x_2 + \gamma}}$**.



### 4. The Intermediate Sum ($u$)

The fourth equation is $u = h_1 - 3h_2 + \delta$.

* **For $h_1$:** The coefficient is $1$, so **$\frac{\partial u}{\partial h_1} = 1$**.
* **For $h_2$:** The coefficient is $-3$, so **$\frac{\partial u}{\partial h_2} = -3$**.
* **For $\delta$:** This is a solo parameter with no multiplier, so **$\frac{\partial u}{\partial \delta} = 1$**.

### 5. The Output Prediction ($\hat{p}$)

The final equation is $\hat{p} = \frac{e^u}{e^u + 1}$.

* This is the **Sigmoid** function.
* A very helpful property of the Sigmoid function is that its derivative is simply the function itself multiplied by (1 - itself).
* **$\frac{\partial \hat{p}}{\partial u} = \hat{p}(1 - \hat{p})$**.

---

### How they connect

Think of the **Chain Rule** like a series of gears. To see how $\alpha$ affects the final Loss ($L$), you multiply the "gears" together:


$$\frac{\partial L}{\partial \alpha} = \underbrace{\frac{\partial L}{\partial \hat{p}}}_{\text{Loss Change}} \cdot \underbrace{\frac{\partial \hat{p}}{\partial u}}_{\text{Output Change}} \cdot \underbrace{\frac{\partial u}{\partial h_1}}_{\text{Sum Change}} \cdot \underbrace{\frac{\partial h_1}{\partial z}}_{\text{Trig Change}} \cdot \underbrace{\frac{\partial z}{\partial \alpha}}_{\text{Input Change}}$$
