# Describe the fundamental principles of ML
Machine learning is a discipline within Artificial Inteligence that use algorithms and computers to make predicitions and learn patterns.

# Define the difference between unsupervised and supevised learning techniques.
There are 3 types of machine learning

1. Unsupervised - given unlabeled data, learn about the data; patterns -> make a prediction.
2. Supervised - given labeled data, make a prediction of the output given an input.
3. Reinforcement - self learning through making mistakes.
