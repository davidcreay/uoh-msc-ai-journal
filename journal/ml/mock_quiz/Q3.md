### Question 3
**Check all statements about the Neural Gas algorithm that are correct.**

* The effective learning rate for each codevector depends on its rank, in terms of closeness to the currently selected data point.
* The initial edge set contains all possible edges.
* A codevector will never be moved towards a data point if there exists another data point such that it is closer.
* Increasing the maximal edge age results in a graph with more edges.

**Answer & Explanation:**
The correct statements are:
* **✅ The effective learning rate for each codevector depends on its rank, in terms of closeness to the currently selected data point.**
* **✅ Increasing the maximal edge age results in a graph with more edges.**

**Why the others are false:**
* ❌ *The initial edge set contains all possible edges.* **Reason:** The algorithm starts with zero edges (an empty graph). It dynamically builds connections (edges) between codevectors during training using a process called Competitive Hebbian Learning. It connects the "winner" and the "runner-up" for each data point.
* ❌ *A codevector will never be moved towards a data point if there exists another data point such that it is closer.* **Reason:** In Neural Gas, when a data point is selected for a training step, *every single codevector* in the network is moved towards that specific point. The size of their movement is dictated strictly by their rank distance to that point, regardless of where the rest of the unselected dataset is located.

**Key Concept Summary:**
Neural Gas is unique because it is "softly competitive." Instead of a "winner-takes-all" approach where only the closest codevector updates, Neural Gas ranks all vectors from closest to furthest. The closest vector takes a large step toward the data, the second closest takes a smaller step, and so on, creating an exponential decay in the learning rate based on rank. Additionally, it prunes old edges that haven't been reinforced recently; increasing the maximum age allows more edges to survive, resulting in a denser topology.
