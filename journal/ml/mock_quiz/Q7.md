### Question 7

Consider the following candidate functions for computing distances between vectors `x` and `y`:

```r
f1 <- function(x, y) {
  d <- x - y;
  return(sqrt(sum(d * d)));
}

f2 <- function(x, y) {
  return(f1(x, x));
}

f3 <- function(x, y) {
  i <- max(1L, as.integer(floor(length(x) / 2)));
  return(f1(x[1L:i], y[1L:i]));
}

f4 <- function(x, y) {
  return(sum(x * y) / f1(x, x) / f1(y, y));
}
```

Determine which of the functions defined above implement a distance in the strict mathematical sense, and check all correct statements below:

* `f3` is a valid distance.
* `f2` is a valid distance.
* `f1` is a valid distance.
* `f4` is a valid distance.
